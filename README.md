## Overview:
A simple API for the recognition of images using AWS Rekognition on the back-end. The API stores an image, does image recognition and labeling on it and returns results to the user in two ways, with a provided `callback` or a using the `Get Blob Labels` endpoint.


<br><br><b>This is an updated version from [image-recognition-api](https://github.com/kfrawee/image-recognition-api)</b>:
- Applying code best practice.
- Applying better project structure.
- Applying better logging and error handeling.
